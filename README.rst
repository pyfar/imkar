* Free software: MIT license


Getting Started
===============

Check out `read the docs`_ for the complete documentation. Packages
related to pyfar are listed at `pyfar.org`_.

Installation
============

Use pip to install imkar

.. code-block:: console

    $ pip install imkar

(Requires Python 3.8 or higher)
